## RandomCall 点名器
作为化学课上取代万恶之源失败的点名器，是时候公开它的源代码了。

### 运行方法
1. 编译

    运行 `build.bat`

    编译的产物在 `dist/randomCall` 里

2.  Debug mode

    打开 CMD
```
python randomCall.py
```
